Источники цен конкурентов
Есть 2 источника:

(1) Pharm Center API
Получаем данные через API (токен уже есть)
В ответе есть поле id — это SKU
(2) Provisor (парсинг через токен)
Используется авторизация (логин/пароль → получаем токен)
В данных есть:
id
distributorGoodsId → это и есть SKU
3. Сопоставление данных
Логика:

SKU =

Excel: Материал (на фронте сейчас SKU)
Pharm Center: id
Provisor: distributorGoodsId
Все три источника должны матчиться по SKU.


ПРАВКИ=>:
НАЗНАЧЕНИЕ ПРАЙС ЛИСТА:
1) Вместо дублирования (Наименования + Регион) пишем Клиента Например: Аптека со Склада
2) Возможность открыть Прайс и показывать Данные: Идентификатор провизора,	Идентификатор Филиала(Прайса),	Наименование,	РЕГ НОМЕР,	Наименование у ДИСТРИБЬЮТЕРА,	Идентификатор Дистрибьютера,	Цена Дистрибьютера,	Остаток,	Упаковок.
3) Возможность Экспортировать Прайс в excel/csv

СФОРМИРОВАННЫЕ ПРАЙС ЛИСТЫ:
1) Добавляем колону ЛОГ(примечение), под какую цену встала наша модельная цена
2) Модельную цену выстраиваем на основе выбранных Прайсов во вкладке - НАЗНАЧЕНИЕ ПРАЙС ЛИСТА
3) Возможность Экспортировать Таблицу Модельная цена в excel/csv
